# Terp TrailFinder

## Description
Allows users to look up trails based on country (US or Canada), state/province, city, and activity type. These trails are saved immediately, which can be seen and deleted.

## APIs Information
[Trail API](https://rapidapi.com/trailapi/api/trailapi)

## Deployed App Link
https://terp-trailfinder.onrender.com/

## Getting Started
Follow the instructions below to download a local version of the project for developing and testing purposes.

### Prerequisites
* Node.js version 22.22.2 or higher.
* npm version 11.13.0 or higher.

### Installation
1. Clone this repository:
```
git clone https://github.com/hl709/Terp-TrailFinder
```
2. Navigate to the client folder:
```
cd Terp-TrailFinder/client
```
3. Install the necessary dependencies using npm:
```
npm install
```

### Running the Development Server
1. To start the client in a local development server, run the following command:
```
npm run dev
```

This will start the server at http://localhost:5173. You can now view the site in your browser.

2. To start the server, run the following commands to navigate to the server folder:
```
cd ..
cd Terp-TrailFinder/server
```

3. Run the following command to start the server:
```
node server.js
```
